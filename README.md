# Clipboard landing page - Frontend Mentor 

## Olá! 👋

Este repositório é destinado a um desafio do [Frontend Mentor](https://www.frontendmentor.io/home) que desenvolvi. Você pode conferir o desafio original [aqui](https://www.frontendmentor.io/challenges/clipboard-landing-page-5cc9bccd6c4c91111378ecb9) e meu resultado final [aqui](https://keilacortes.github.io/clipboard-landing-page/).

## O desafio

O objetivo foi criar uma landing page seguindo o design fornecido e deixá-lo o mais fiel possível à proposta. Além disso, o site deve oferecer uma experiência otimizada em diferentes tamanhos de tela e mostrar os estados de hover em todos os elementos interativos da página.

### Processo e resultado

Após visualizar os designs e planejar como iria estruturar o HTML e CSS, comecei escrevendo o HTML e depois fui adicionando os estilos de cima para baixo. Você pode conferir o resultado nas screenshots e nos links a seguir:
